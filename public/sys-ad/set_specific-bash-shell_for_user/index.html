Article bin/bash shell - change shell
Linux bash shell

If user account is set to use sh instead of bash some usual commands may not work. * Does up arrow showing the last command? * Does left arrow allow to move cursor? * Does TAB autocompletes commands? One of these points might showcase an incorrect user profile for shell. To fix it chsh -s /bin/bash The rootcause can be a badly initialised default config for user accounts.
